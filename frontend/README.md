# BoardWall Frontend

React + TypeScript frontend for BoardWall task management system.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features

- Authentication (Login/Register)
- Dashboard with statistics
- Project management
- Task tracking with Kanban boards
- Time tracking
- Reports and analytics
- User settings

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Query
- React Hook Form
- Zustand
- Recharts
- Axios
